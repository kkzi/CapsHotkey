# CapsHotkey
Use capslock key do more

## hotkey mappings
| Key | Desc|
|:---|:----|
| [Capslock] + A | Home |
| [Capslock] + B | Backford |
| [Capslock] + C | Delete from cursor to line end |
| [Capslock] + E | End |
| [Capslock] + F | Forward |
| [Capslock] + H | Left |
| [Capslock] + J | Down |
| [Capslock] + K | Up |
| [Capslock] + L | Right |
| [Capslock] + N | Down |
| [Capslock] + O | Insert blank line before current |
| [Capslock] + P | Paste |
| [Capslock] + S | Selete current line |
| [Capslock] + U | Delete word backforward |
| [Capslock] + Y | Copy |
| [Capslock] + / | Show Help |
